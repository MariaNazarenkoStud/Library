# Каталог книг

Десктопний застосунок для управління особистою колекцією книг, розроблений на Java з використанням Swing.

## Функціонал

- Додавання, редагування та видалення книг з каталогу
- Пошук книг за назвою та за автором
- Фільтрація книг за жанром
- Збереження та завантаження каталогу з бінарного файлу
- Зручний графічний інтерфейс

## Структура проєкту

```
Library/
├── src/
│   ├── Publication.java            # Абстрактний базовий клас для публікацій
│   ├── Book.java                   # Клас книги (розширює Publication)
│   ├── BookNotFoundException.java  # Виняток при відсутності книги
│   ├── Catalogue.java              # Логіка каталогу та робота з файлами
│   └── BookGUI.java                # Графічний інтерфейс (Swing)
├── .gitignore
├── LICENSE
└── README.md
```

## Вимоги

- Java 17 або вище

## Запуск

1. Компіляція:
   ```bash
   javac -d out src/*.java
   ```

2. Запуск:
   ```bash
   java -cp out BookGUI
   ```

## Стандарти документування

Усі публічні класи та методи **обов'язково** мають містити Javadoc-коментарі:

- **Класи** — опис призначення, архітектурної ролі, приклад використання (`<pre>{@code ... }</pre>`)
- **Методи** — опис поведінки (не реалізації); теги `@param`, `@return`, `@throws`
- **Архітектурні рішення** — пояснення *чому* (inline-коментарі для неочевидної логіки)
- **`@see`** — посилання на пов'язані класи для навігації в HTML-документації

Генерація HTML-документації:
```bash
javadoc -d docs/javadoc -sourcepath src -encoding UTF-8 src/*.java
```

Повна інструкція: [`docs/generate_docs.md`](docs/generate_docs.md)
